# 🚨 Smart Emergency System

A full-stack web application designed to enhance personal safety by enabling users to send real-time SOS alerts with live location to their emergency contacts.

---

## 🔥 Features

- 🔐 User Authentication (Register & Login)
- 📇 Add Emergency Contacts
- 📍 Live Location Tracking (Geolocation API)
- 🚨 One-click SOS Alert
- 📲 SMS Alerts using Twilio
- 🖥️ Separate Login & Dashboard UI

---

## 🛠️ Tech Stack

### Frontend:
- HTML, CSS, JavaScript

### Backend:
- Node.js
- Express.js

### Database:
- MongoDB

### APIs & Services:
- Twilio API (SMS Alerts)
- Google Maps (Location Embed)

---

## ⚙️ How It Works

1. User registers and logs in
2. Adds emergency contacts
3. Clicks SOS button
4. System fetches live location
5. SMS alert is sent to all contacts with Google Maps link
